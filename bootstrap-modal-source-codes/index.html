Bootstrap Modal Windows

We can quickly append functionality for modal windows in HTML. This demo will load windows from div elements found in the page markup.

The first button is a generic window, and the 2nd allows us to reset the body background color.

A very special thanks out to Twitter Bootstrap for this sweet framework, and to Net Tuts+ for the awesome radio buttons.
